# REALITY GLITCH OS

A simulated decommissioned government terminal containing a digital anomaly.

## Features

- Terminal-style interface
- Clickable apps
- Persistent entity memory
- Learning behavior
- Mood system
- Text-based apps
- Fake internet
- System instability progression
- LocalStorage persistence

## Run

Open `index.html` in browser. No build required.

## Apps

Clickable words:

- FILES
- MONITOR
- NOTES
- MEDIA
- BROWSER
- CHAT

### Usage

- Notes: `writeNote('text')`
- Chat: `talk('message')`
- Browser: `search('query')`
